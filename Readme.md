# 3rd Grade NGSS Home Learning Curriculum

A comprehensive, home-implementable science curriculum aligned to Washington State's 3rd Grade Next Generation Science Standards (NGSS). Designed for one student working with one parent/teacher.

✅ **Question Scripts** - Know exactly what to ask and when  
✅ **PBS Resources Integrated** - Free, high-quality videos and materials

✅ **NGSS-Aligned** - Meets all Washington State standards

## Quick Start

### To Use the Completed Physical Science Guide:

1. **Review the guide**: [PhysicalScience/PhysicalScienceGuide.md](PhysicalScience/PhysicalScienceGuide.md)
2. **Print PDF templates**: Download from [PhysicalScience/Templates/](PhysicalScience/Templates/)
3. **Gather household materials** (listed in each section's Setup)
4. **Watch prep videos** (links provided in guide)
5. **Begin Session 1** (typically 60-75 minutes)

### To Generate Remaining Sections:

If you want to create the Life Science, Earth Science, or Environmental Science guides:

1. **Review the master prompt**: [Prompts/PROMPT_Template_for_Remaining_Sections.md](Prompts/PROMPT_Template_for_Remaining_Sections.md)
2. **Use with AI assistant** (Claude, GPT-4, etc.)
3. **Follow the Quick Start Guide**: [Prompts/Quick_Start_Guide.md](Prompts/Quick_Start_Guide.md)

**Expected Generation Time**: 5-7 hours for all three remaining sections

## Implementation Timeline

### Year-Long Pacing (Recommended)
- **Weeks 1-5**: Physical Science (Forces and Motion)
- **Weeks 6-12**: Life Science (7 weeks for 8 standards)
- **Weeks 13-16**: Earth Science (with ongoing weather observations)
- **Ongoing**: Environmental & Sustainability projects throughout year

**Total**: 57-79 hours of instruction across school year

### Intensive Pacing (Accelerated)
- 3-4 sessions per week
- Complete in one semester
- Requires dedicated time blocks
