## Quick Start

### To Use the Curriculum:

**Option 1: Start with Physical Science**
1. **Review the guide**: [PhysicalScience/PhysicalScienceGuide.md](PhysicalScience/PhysicalScienceGuide.md)
2. **Print PDF templates**: Download section-specific templates
3. **Gather household materials** (listed in each section's Setup)
4. **Watch prep videos** (links provided in guide)
5. **Begin Section 1** (typically 60-75 minutes)

**Option 2: Choose by Interest**
- Student loves animals/nature? Start with **Life Science**
- Fascinated by weather? Start with **Earth Science**
- Wants to take environmental action? Start with **Environmental Science**
- All sections are independent and can be completed in any order

**Option 3: Integrated Approach**
- Follow the recommended year-long pacing (see below)
- Run Environmental Science projects alongside other sections
- Coordinate weather observations (Earth Science) with other learning

### Getting Started Checklist:

- [ ] Review overview guide for your chosen section
- [ ] Print PDF templates for first few sessions
- [ ] Gather household materials
- [ ] Create free PBS LearningMedia account
- [ ] Set up science notebook with student
- [ ] Watch any prep videos for first session
- [ ] Schedule first session on calendar

## Implementation Timeline

### Year-Long Pacing (Recommended)

**Fall Semester (Weeks 1-18)**:
- **Weeks 1-5**: Physical Science (Forces and Motion)
- **Weeks 6-12**: Life Science (8 sections)
- **Weeks 13-18**: Earth Science (with daily weather observations starting week 13)
- **Ongoing**: Environmental Science Project 1 (weeks 8-12)

**Spring Semester (Weeks 19-36)**:
- **Weeks 19-22**: Continue/review Physical & Life Science concepts
- **Weeks 23-30**: Environmental Science Project 2 (Home Grounds Investigation)
- **Weeks 30-36**: Environmental Science Project 3 (Personal Action - continues beyond semester)
- **Integration**: Weather observations, seasonal connections

**Total**: 59-82 hours of core instruction + ongoing environmental projects

### Intensive Pacing (One Semester)

Complete core curriculum in 16-20 weeks:
- 3-4 sessions per week
- Focus on Physical, Life, and Earth Science
- Add Environmental Science projects as extensions
- Requires dedicated 2-3 hour blocks

### Flexible/Interest-Led

- Let student choose section order
- Spend extra time on topics of interest
- Add extension activities
- Integrate with other subjects (writing, art, math)
- Environmental Science spans entire year as ongoing engagement

## Frequently Asked Questions

**Do I need to complete sections in order?**
- No! Sections are independent. Start with student interest.
- Recommended order: Physical ‚Üí Life ‚Üí Earth ‚Üí Environmental (or Environmental ongoing)
- But any order works.

**What if we don't have certain materials?**
- All materials are household items
- Substitutions are suggested throughout
- Most investigations can be adapted

**Can this be used in a co-op or small group?**
- Designed for 1:1, but adaptable
- Would need to adjust some investigations
- PDF templates work for small groups

**How does this compare to traditional curriculum?**
- More hands-on and inquiry-based
- More depth, less breadth
- Student-driven investigation
- Portfolio vs. traditional tests

**Is this aligned to standards in other states?**
- Yes! NGSS is used in many states
- WA standards = NGSS + ESE additions
- Core content applicable anywhere
